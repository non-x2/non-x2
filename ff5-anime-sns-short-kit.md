## 0. このキットについて

FF5アニメ試作を **縦型ショート動画（SNS拡散用）** に落とし込んだ制作設計図。  
本リポジトリ環境では映像レンダリング／音声合成は行えないため、**外部の生成AIに流し込めば完成する形**まで作り込んでいます。

### スペック
| 項目 | 値 |
| --- | --- |
| アスペクト比 | **9:16（縦型）** |
| 解像度 | 1080 × 1920 px |
| 尺 | **30秒版**（本命）＋ **15秒版**（超圧縮） |
| フレームレート | 30fps（i2vは24/30どちらでも可） |
| 想定媒体 | TikTok / Instagram Reels / YouTube Shorts |
| 音声 | 音なし自動再生が前提 → **テロップ必須**。音ありでBGM＋効果を強化 |

### セーフゾーン（重要）
- 縦型は**上下にUI（ユーザー名・キャプション・ボタン）が被る**。
- 重要なテロップ・被写体は**中央の天地80%**に収める（上12%・下20%は避ける）。

## 1. SNSショート 設計原則

1. **最初の2秒で掴む（フック）**：いきなり“画”の強いカット＋煽りテロップ。説明は後回し。
2. **テンポは本編の倍速**：1カット1〜2.5秒。間延び厳禁。
3. **テロップ・ファースト**：音を切っていても意味が伝わるよう、全カットに短い字幕。
4. **ループ設計**：ラストを冒頭に繋げて“もう一回見たくなる”作り（離脱率低下＝アルゴリズム評価UP）。
5. **縦の構図**：横長の戦闘を無理に入れず、キャラのアップ・煽り・俯瞰など縦映えする画を選ぶ。
6. **1メッセージ**：このショートで伝えるのは「“継承”の感動」一本に絞る。

## 6. テロップ・音声・BGM

### テロップ仕様
- フォント：太めのゴシック（視認性重視）、白文字＋濃い縁取り or 半透明帯。
- 出現：各カット頭で**素早くポップ**（タイプ表示は遅いので避ける）。
- 位置：中央〜やや下（ただし下20%のUI被りゾーンは避ける）。

### 音声（任意・音あり視聴用）
- ナレーション最小限。本キットでは**クルルの一言**のみ採用：
  - S9 クルル：「うん──いくよ。」 *Dir: 涙をこらえつつ前を向く。芯のある少女声。* / EN: "Yeah... let's go."
- TTS：VOICEVOX / にじボイス / ElevenLabs 等。声設計は[設定資料集](./ff5-anime-movie-and-worldbook.md)準拠。

### BGM
- 「親愛なる友へ」系の感動アレンジを30秒に再構成。S7でブレイク→S8で立ち上げる緩急。
- ※SNSはトレンド音源を当てると伸びやすいが、原曲・公式音源の権利に注意（非営利でも要確認）。

## 8. 制作フロー（最短手順）
1. §4のプロンプトで各カットの**9:16静止画**を生成。
2. §5のプロンプトで**静止画→動画**化（各1〜3秒）。
3. §6のテロップを**編集ソフト（CapCut等）で全カットに付与**（CapCutはShorts向け機能が充実）。
4. BGM・効果音・（任意で）クルルの声を合わせ、**30秒/15秒に尺合わせ**。
5. ループ用にラスト0.3秒へ冒頭カットを差し込み、書き出し（1080×1920 / mp4）。
