## 4. 制作規約（次の担当者は必ず踏襲）

### 脚本フォーマット（`ff5-anime-ep0X-screenplay.md`）
- 構成：`0.話数情報`表 → `アバン` → `OP` → `Aパート` → `アイキャッチ` → `Bパート` → `ED → 次回予告` → `制作メモ`
- 記法：**柱**＝`◯ 場所・時間`／（ト書き）＝情景・芝居／`**名前**「セリフ」`
- 尺：各約23分。冒頭に非営利の注記、関連リンク（前話・企画書）を付ける。
- 1話＝1ファイル。ファイル名は `ff5-anime-ep0X-screenplay.md`（2桁ゼロ埋め）。

### README
- `### 🎬 創作実験` の脚本リスト（📜配下）に、新しい話を追記する。

### Git / PR フロー
1. **バッチ単位の新ブランチ**を main から作成：`git checkout main && git pull origin main` → `git checkout -b claude/ff5-anime-epNN-MM main`。
   - 理由：PRは **squashマージ** 運用のため rolling ブランチは毎回分岐する。バッチごとに main から切れば常にクリーンで、merge同期も `git reset --hard`（環境のセーフガードで拒否される）も不要。
2. バッチの数話＋README＋本HANDOFFをまとめてコミット → `git push -u origin <branch>`。
3. PR作成〜マージ（`gh` CLI が確実。non-x2 で認証済み）：
   - `gh pr create --base main --head <branch> --title "…" --body "…"`
   - `gh pr merge <PR#> --squash --delete-branch`（**push＝公開なので、のんさん確認後に実行する**）
   - CIは無い（ステータスチェック0件）。まれに「Base branch was modified」→ PR状態を再取得して再マージで成功する。
4. コミット末尾フッター：`Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>`。PR本文末尾に `🤖 Generated with [Claude Code]` を付す。

### 補助ツール
- **Gamma**（`mcp__Gamma__generate`）で絵コンテ/プレゼンの画像付きスライドを生成可能。
  - presentation形式は **numCards最大10**、**9:16非対応（fluidになる）**。縦構図はimageOptions.styleに "vertical 9:16" を入れて対応。
  - 既存Gamma：ピッチ `4znD6QBhK6NNoPHFp4A4S` ／ 第1話絵コンテ `cc93a9blitvt2x1` ／ SNS絵コンテ `ynDq6zKTyksoUI0eNygq3` ／ ジョブあるある `b9aDSZSF1WQj2KLP4LGNN`。

## 5. 物語の継続性メモ（脚本の一貫性のため厳守）

### 確定設定（原作準拠）
- **バッツ・クラウザー**：主人公。父は暁の4戦士**ドルガン**（第1世界に残った）。相棒チョコボ=ボコ。→ 出自判明は第9〜10話で。
- **レナ**：タイクーン王女。慈愛と決意。
- **ファリス（サリサ）**：海賊頭目。**実はレナの実姉**で行方不明の王女。海竜シルドラの相棒。※姉妹判明は今後の見せ場として温存可。
- **ガラフ**：暁の4戦士。**第10話で全力でエクスデスに挑み、力尽きてなお立ち、孫クルルに意志と力を託して死す**（シリーズ屈指の名場面）。
- **クルル**：ガラフの孫。第10話以降、新たな光の戦士として継承。
- **暁の4戦士**：ドルガン／ガラフ／ゼザ・マティアス・サーゲイト／ケルガー・ヴロンデット。30年前にエクスデス封印のため世界を2つに分割。
- **エクスデス**：4クリスタルが引き寄せた邪悪な意志が大樹に宿った暗黒魔道士。→ 第8話で完全復活済み。
- **エヌオー**：1000年前に“無”を操った伝説の魔道士（第16話で前史を語る）。
- **ネクロフォビア**：次元の狭間の門番。**ここでギルガメッシュが再登場し味方として散る**（第21話）。
- **ネオエクスデス**：無を取り込んだ最終形態（第23話）。撃破後、4クリスタルの精霊が再生し世界を守る。
- **世界統合**：第19話で2つの世界が1つに戻り、ザ・ヴォイド（無）が出現。

### 本企画オリジナル要素（踏襲する脚色）
- **顔のある敵幹部**：第1〜4話でクリスタル崩壊を担うアニメ用キャラ（牽引役）。
- **ジョブ獲得演出**：「クリスタルの欠片が体に灯る」。成長＝生き方の選択（企画書§8の作画文法に接続）。
- 既出の伏線：第8話でバッツが「ドルガン」の名に反応 → 第9〜10話で回収。第6話ギルガメッシュ「もっと強くなって戻ってくる」→ 第12話・第21話で回収。

### キャラの声・口調
- バッツ：飄々・軽口、芯は熱い（「乗りかかった船だ」）。
- ガラフ：渋く温かい、コミカルもいける。英雄の重み。
- ファリス：べらんめえ調の男言葉、内に孤独。
- レナ：丁寧で優しい、芯は強い。
- クルル：明るく強情。
