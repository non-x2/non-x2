# 📐 スタイルガイド — 選び方特捜部の「型」

すべてのテンプレートの土台となる、チャンネルの**思想・トーン・編集原則**です。
台本やサムネに迷ったら、まずここに立ち返ってください。

---

## 🎯 チャンネルの約束（コンセプト）

> **視聴者の「買い物の意思決定」を、代わりにやりきる。**

ホームセンターに並ぶ無数のモノの中から、「結局どれを買えばいいの？」に対して、
**根拠のある結論**を、**忖度なく**、**初心者にも分かる言葉**で届けます。

視聴者が動画を見終わったときに残る感情はこれ：
**「もう迷わない。これを買えばいい、と分かった。」**

---

## 🧭 5つの編集原則

1. **結論ファースト**
   「で、どれがいいの？」を引っ張らない。冒頭で結論の方向性を予告し、本編で根拠を示す。

2. **忖度なしの比較**
   メーカーや価格に媚びない。高い＝正解ではない。**用途とタイプ別に「あなたにとっての最適解」**を出す。デメリットも必ず言う。

3. **専門用語はかみ砕く**
   「トルク」「番手」「ルーメン」などは、必ず一言で意味と「だから何が変わるか」を添える。

4. **現実的な選択肢に絞る**
   ホームセンター（やネット通販）で**実際に買えるもの**を扱う。入手性・実売価格を重視。マニアックすぎる選択肢は出さない。

5. **失敗を先回りで潰す**
   「初心者がやりがちな失敗」「買ってから後悔するポイント」を必ず入れる。これが信頼の核。

## 🗣 トーン＆ボイス

| 項目 | 方針 |
|------|------|
| 人称・語り口 | 親しみやすい「です・ます」。たまに「正直」「ぶっちゃけ」で本音感を出す |
| 立ち位置 | 上から教えるのではなく、**視聴者の隣で一緒に選ぶ相棒**。「特捜部」=徹底的に調べてあげる係 |
| テンポ | 1ブロック＝1メッセージ。冗長な前置きはカット。間延びさせない |
| 誠実さ | 分からない/個人差がある点は正直に言う。断定できる所は言い切る |
| ユーモア | 入れてOK。ただし情報の正確さを犠牲にしない範囲で |

### ❌ NG例（やらないこと）
- 「人によります」で終わらせて結論を逃げる
- メリットだけ並べてデメリットを隠す
- スペックの数字を読み上げるだけで「だから何が変わるか」を言わない
- 在庫が無い/手に入りにくいニッチ商品を勧める
- 煽りサムネと中身が一致しない（クリックベイト）

## ⏱ 尺・構成の基準

| 項目 | 目安 |
|------|------|
| 動画の長さ | **8〜15分**（選定基準が多いテーマは長め、単品比較は短め） |
| 冒頭フック | **最初の15秒**で「悩み」と「結論予告」を提示（[library/フック集](library/フック集.md)） |
| 選定ポイント数 | **3〜5個**に絞る（多すぎると覚えられない） |
| おすすめ提示 | 価格帯別 or 用途別で**2〜4商品**。「全部入り」は避ける |
| CTA | 終盤で次の動画へ導線（[library/CTAパターン集](library/CTAパターン集.md)） |

詳しい構成は [templates/03_構成・台本.md](templates/03_構成・台本.md)、
維持率テクは [library/視聴維持率テクニック.md](library/視聴維持率テクニック.md) を参照。

## 🎨 サムネの基本方針

- **3秒で「何の選び方か」が分かる**こと（商品カテゴリ＋ベネフィット or 数字）
- 文字は**13文字以内**を目安に、最大2行。スマホで読める大きさ
- 「結論」or「失敗回避」を訴求軸にする（例：`電動ドライバー どれ買う？` / `9割が知らない選び方`）
- 詳細は [templates/04_サムネイル設計.md](templates/04_サムネイル設計.md)
