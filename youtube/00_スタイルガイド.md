# 📐 スタイルガイド — 選び方特捜部の「型」

すべてのテンプレートの土台となる、チャンネルの**思想・トーン・編集原則・制作前提**です。
台本やサムネに迷ったら、まずここに立ち返ってください。

---

## 🎯 チャンネルの約束（コンセプト）

> **DIY・農業（家庭菜園）・水回り・暮らしまわりで「どれを選べばいいの？」と迷ったとき、
> 「3つのポイント」で整理して解決する選び方ガイド。** 🔍

「電動工具のスペックの見方がわからない」「家庭菜園、何から揃えればいい？」「水道のパッキン、どれ買えば合うの？」——
そんな**中年家庭層の「自分でやってみたい」気持ち**を応援します。

立ち位置は **「調べた・整理した・伝える、それだけ」**。
公開情報をもとに徹底的に調べ、迷いを3つのポイントに整理して届けるチャンネルです。

視聴者が見終わったときに残る感情はこれ：
**「これ気になってた！ もう迷わない、これを選べばいい、と分かった。」**

---

## 🧭 5つの編集原則

1. **「3つのポイント」を厳守する** 🔍
   選び方は**必ず3つのポイントに整理**する。これがチャンネルの看板フォーマット。多くても少なくてもブレる。

2. **結論ファースト**
   「で、どれを選べばいいの？」を引っ張らない。冒頭で「3つのポイントで解説します」と予告し、本編で根拠を示す。

3. **専門用語はかみ砕く**
   「トルク」「番手」「呼び径」「電圧」などは、必ず一言で意味と「だから何が変わるか」を添える。中年家庭層が迷わない言葉で。

4. **調べた根拠を示す**
   「調べた・整理した・伝える」が軸。憶測でなく公開情報（メーカー公式・販売情報）に基づく。出典を残す。

5. **免責で安全に届ける**
   このチャンネルは**個人の調査**。誤情報の可能性に触れ、**最終判断は専門家へ**。特に**水回り工事・農業使用・電気工事は有資格者へ**と必ず案内する（詳細は [library/概要欄定型文.md](library/概要欄定型文.md)）。

### ❌ NG例（やらないこと）
- ポイントを4つ5つに増やして散漫にする（3つに絞る）
- 「人によります」で終わらせて結論を逃げる
- メリットだけ並べてデメリットを隠す
- スペックの数字を読み上げるだけで「だから何が変わるか」を言わない
- 工事・施工・薬剤使用を「自分でやればOK」と断定する（必ず有資格者・専門家への案内を添える）
- 煽りサムネと中身が一致しない（クリックベイト）

## 🎙 制作前提（重要・このチャンネル固有）

| 要素 | 仕様 |
|------|------|
| **音声** | **VOICEVOX「玄野武宏」** による合成音声ナレーション。**人物は登場しない**（顔出し・実演トークなし） |
| **ビジュアル** | **AI生成画像を活用**。実物撮影中心ではなく、AI画像＋図解・テロップで見せる |
| **台本の性質** | ナレーション主体。司会者の「自己紹介」ではなく**チャンネル紹介ナレーション**で入る |
| **収益** | 概要欄に**アフィリエイトリンク**を含む場合あり（表記必須） |

### 🗣 VOICEVOX向け 台本ルール
合成音声で自然に・正確に読ませるための約束：
- **一文を短く**。長い文は句読点で割り、息継ぎのリズムを作る
- **難読漢字・専門用語にはふりがな/読み注記**（例：呼び径＝よびけい、捗る→「はかどる」とひらがな化も検討）
- **数字・型番の読み方を明記**（例：「13mm」→「じゅうさんミリ」、「P-20」→「ピー　にじゅう」）
- **イントネーションがおかしくなりがちな語**はメモを残し、収録時に調整（VOICEVOXのアクセント編集）
- 「！」「…」など記号で**間と感情**を作る。棒読み回避
- 同音異義語は文脈で誤読しないよう言い換える

## 🗣 トーン＆ボイス

| 項目 | 方針 |
|------|------|
| 語り口 | 親しみやすい「です・ます」。中年家庭層に語りかける落ち着いたトーン |
| 立ち位置 | 上から教えない。**一緒に調べてあげる「特捜部」**。「調べた・整理した・伝える」係 |
| テンポ | 1ブロック＝1メッセージ。冗長な前置きはカット。テロップでリズムを作る |
| 誠実さ | 個人調査であること・誤情報の可能性を隠さない。断定できる所は言い切る |
| 絵文字 | 🔍 を基調（特捜＝調べる）。⚠️ は注意喚起に |

## ⏱ 尺・構成の基準

| 項目 | 目安 |
|------|------|
| 動画の長さ | **8〜15分**（テーマの広さに応じて。短めの単品テーマもOK） |
| 冒頭フック | **最初の15秒**で「悩み」＋「3つのポイントで解説」を予告（[library/フック集.md](library/フック集.md)） |
| 選定ポイント数 | **必ず3つ**（看板フォーマット） |
| おすすめ提示 | タイプ別／用途別で**2〜4点**。「全部入り」は避ける |
| 注意（免責） | 工事・施工・薬剤系テーマでは**動画内でも一言**＋概要欄に定型文 |
| CTA | 終盤に「**これ気になってた！が見つかったら登録お願いします** 🔍」 |

詳しい構成は [templates/03_構成・台本.md](templates/03_構成・台本.md)、維持率テクは [library/視聴維持率テクニック.md](library/視聴維持率テクニック.md) を参照。

## 🎨 サムネの基本方針

- **3秒で「何の選び方か」が分かる**（ジャンル＋「3つのポイント」or ベネフィット）
- 文字は**13文字以内**目安、最大2行。スマホで読める大きさ
- **AI生成画像**で対象物を分かりやすく。🔍モチーフでチャンネルらしさ
- 訴求軸は「結論」or「失敗回避」or「3つのポイント」（例：`水道パッキン どれ選ぶ？` / `失敗しない3つのポイント`）
- 詳細は [templates/04_サムネイル設計.md](templates/04_サムネイル設計.md)
