# 📈 視聴維持率テクニック

> 「最後まで見てもらう」ための、編集・話法のチェック集。台本（[../templates/03_構成・台本.md](../templates/03_構成・台本.md)）と
> 編集時（[../templates/06_公開前後チェックリスト.md](../templates/06_公開前後チェックリスト.md)）に見返してください。
> このチャンネルは **VOICEVOXナレーション＋AI生成画像**が前提なので、それ向けの施策も含みます。

---

## 🚪 冒頭の離脱を防ぐ（最初の30秒）
- [ ] 長い挨拶を**フックの後ろ**に回す
- [ ] 15秒以内に「悩み」と「**3つのポイントで解説**」を提示
- [ ] サムネ／タイトルの“約束”をすぐ回収する（裏切らない）
- [ ] 1枚目を動きのある絵・印象的なAI画像にする（退屈な静止から始めない）

## 🔮 先読みの「予告」を仕込む
視聴者に「最後まで見る理由」を持たせる。
- 「特に3つ目のポイントは、知らないと損するので最後まで見てください。」
- 「動画の最後に、タイプ別の“結論まとめ”を出します。」

## 🧱 1ブロック＝1メッセージ
- [ ] 1つの話題を長く引っ張らない。テンポよく次へ
- [ ] 同じことを繰り返さない（要点は1回、強く）
- [ ] **3つのポイント**で進捗が見えるので、番号（①②③）を画面に常時/節目で出す

## 🎤 VOICEVOXナレーションの工夫
- [ ] 一文を短く。長い文は区切ってテンポを作る
- [ ] 棒読みにならないよう「！」「…」で抑揚・間をつける
- [ ] 難読語・型番の読み間違いを潰す（誤読は一気に離脱を招く）
- [ ] 無音・不自然な間をカット

## 👀 視覚で飽きさせない（AI画像・図解）
- [ ] 数字・比較は**テロップや図解**で見せる（音声だけにしない）
- [ ] 4〜8秒ごとに画に変化（AI画像の切替・ズーム・テロップ）
- [ ] 重要ポイントは効果音＋テロップで“引っかかり”を作る
- [ ] AI画像は破綻のない自然なものを選ぶ

## 🪜 構成で引っ張る
- [ ] 結論を全部先に言い切らず、「根拠は本編で」と適度に引く
- [ ] 「3つのポイント」で進捗を可視化（最後まで見やすい）
- [ ] チャプター（[../templates/05_タイトル・概要欄・SEO.md](../templates/05_タイトル・概要欄・SEO.md)）を設定

## 🔚 終盤の離脱対策
- [ ] まとめは3つのポイントを手短に復習
- [ ] CTAは満足度が高い直後に（[CTAパターン集.md](CTAパターン集.md)）
- [ ] 終了画面で次の動画へ確実に橋渡し

---

## 📊 公開後に見る指標
- **視聴維持率グラフ**：急落箇所＝つまらない/長い → 次回はそこを短く
- **平均視聴時間 / 平均視聴率**：40%超を一つの目安に
- **クリック率(CTR)**：低ければサムネ・タイトル（[../templates/04_サムネイル設計.md](../templates/04_サムネイル設計.md)）を見直す
- 振り返りは [../templates/06_公開前後チェックリスト.md](../templates/06_公開前後チェックリスト.md) に記録
