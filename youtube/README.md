# 🔍 YouTube制作パイプライン — 選び方特捜部

**DIY・農業（家庭菜園）・水回り・暮らしまわり**で「どれを選べばいいの？」と迷ったとき、
**「3つのポイント」で整理して解決する選び方ガイド**動画を、高品質かつ再現性高く量産するための「型（テンプレート群）」です。

対象視聴者は**中年家庭層の「自分でやってみたい」人**。スタンスは「**調べた・整理した・伝える、それだけ**」。

> 制作前提（チャンネル固有）：
> 🎤 **音声＝VOICEVOX「玄野武宏」**（合成音声ナレーション・人物登場なし） ／ 🤖 **ビジュアル＝AI生成画像を活用** ／ ⚠️ **免責・アフィリ表記・クレジットは毎回必須**

> 構成を設計した動画は視聴維持率が大きく上がります（構成あり 平均42% / なし 18%）。
> このパイプラインは、企画から公開後の振り返りまでを一気通貫でテンプレ化し、毎回の「考える手間」をゼロに近づけます。

## 🚀 使い方（1本作るとき）

1. **新しい動画用フォルダ**を作る（例：`works/2026-07_電動ドライバーの選び方/`）。`templates/` の各ファイルをコピーして使う。
2. `00_スタイルガイド.md` に毎回さっと目を通し、**3ポイント厳守・VOICEVOX・AI画像・免責**の軸を思い出す。
3. **① 企画 → ⑥ チェック**の順にテンプレを埋める。前のステップの出力が次の入力になる。
4. 迷ったら `library/`（フック・CTA・維持率テク・**概要欄定型文**）と `examples/`（記入例）を参照。
5. 公開後は `06` の「公開後の振り返り」を記入し、次回の企画にフィードバックする。

### 🤖 AI（Claude等）との使い方

各テンプレは AI に下書きさせる前提でも設計しています。例：

- **企画出し**：「`00_スタイルガイド.md` の型で、◯◯の選び方動画を3案、`01_企画書.md` の形式で。選び方は必ず3ポイントに整理して」
- **台本ドラフト**：「この比較シート（`02`）をもとに、`03_構成・台本.md` の黄金構成でVOICEVOX用ナレーション草案を。AI画像の指示も添えて」
- **概要欄**：「`library/概要欄定型文.md` を使って、この動画の概要欄を作って」

> AIの出力は必ず**スタイルガイドのトーン**と**事実（価格・スペック）**を人間が確認してから採用してください。

## ⚠️ 毎回の必須要素（忘れない）

- **選び方は「3つのポイント」**に整理（看板フォーマット）
- 概要欄＆（該当テーマは動画内でも）**免責文**（個人調査・誤情報可能性・専門家相談／水回り・電気・農業は有資格者へ）
- **アフィリエイト表記**（リンクを貼る場合）
- **クレジット**（🎤 VOICEVOX「玄野武宏」／🤖 AI生成画像を活用）
- 登録CTA「**これ気になってた！**が見つかったら登録お願いします」🔍

定型文は [library/概要欄定型文.md](library/概要欄定型文.md) にまとめてあります。
