## 📝 制作前
- [ ] 台本（[03](03_構成・台本.md)）が最後まで埋まっている
- [ ] **選び方が「3つのポイント」に整理されている**（増えていない）
- [ ] 必要なAI生成画像・図解の一覧ができている
- [ ] 情報の裏取り（[02](02_商品リサーチ比較シート.md) の出典）が済んでいる
- [ ] 免責が必要なテーマか判定済み（水回り/電気/農業）

## 🎤 ナレーション（VOICEVOX）
- [ ] VOICEVOX「玄野武宏」で書き出し
- [ ] **難読漢字・型番の読み間違いがない**（呼び径・mm・型番など）
- [ ] **イントネーション**が不自然な箇所を調整した
- [ ] 一文が長すぎず、間（ま）が自然

## 🖼 ビジュアル・編集
- [ ] AI生成画像に破綻（指・文字崩れ・不自然さ）がない
- [ ] 各ポイントにテロップが入っている
- [ ] 数字・型番・価格のテロップが正確
- [ ] 図解・比較表が分かりやすい
- [ ] 間延び・無音をカットした
- [ ] BGM/効果音の音量がナレーションを邪魔していない
- [ ] 結論が明確に言い切られている
- [ ] ⚠️ 注意（免責）パートが入っている（該当テーマ時）
- [ ] CTA「これ気になってた！」が入っている

## 📤 書き出し設定
- [ ] 解像度1080p以上／フレームレート統一
- [ ] 形式 MP4（H.264 / AAC）
- [ ] 音割れ・無音区間がない

## 🚦 公開前 最終QA
- [ ] サムネ（[04](04_サムネイル設計.md)）を設定／スマホ表示で読める
- [ ] タイトル（[05](05_タイトル・概要欄・SEO.md)）採用案を設定
- [ ] 概要欄に **免責文・アフィリ表記・クレジット（VOICEVOX/AI画像）・CTA** が入っている（[../library/概要欄定型文.md](../library/概要欄定型文.md)）
- [ ] チャプター（0:00開始）を貼った
- [ ] ハッシュタグ・タグを設定
- [ ] 終了画面・カードを設定
- [ ] 再生リスト（ジャンル別シリーズ等）に追加
- [ ] 公開範囲・予約投稿の設定を確認
- [ ] 子ども向け設定／その他の規約項目を確認
- [ ] サムネとタイトルと中身が一致している（クリックベイトでない）

## 📣 公開直後
- [ ] 固定コメント（補足・関連動画リンク）を投稿
- [ ] SNS等で告知（任意）
- [ ] 自分で一度通して再生し、リンク・チャプターの動作を確認
