## 🎥 撮影前
- [ ] 台本（[03](03_構成・台本.md)）が最後まで埋まっている
- [ ] 紹介する実物商品がすべて手元にある
- [ ] 比較実演用の小物・素材を用意した
- [ ] 撮りたいカット（手元アップ・使用シーン・B-roll）をリスト化した
- [ ] マイク音声・明るさ・背景を確認した

## ✂️ 編集中
- [ ] 冒頭15秒のフックが強い（テンポ良く本題へ）
- [ ] 各選定ポイントにテロップが入っている
- [ ] 数字・型番・価格のテロップが正確
- [ ] 間延び・言い淀みをカットした
- [ ] BGM/効果音の音量が声を邪魔していない
- [ ] 比較シーンが視覚的に分かりやすい（表・矢印・並列）
- [ ] 結論が明確に言い切られている
- [ ] CTA（次動画・登録）が入っている

## 📤 書き出し設定
- [ ] 解像度1080p以上（推奨）／フレームレート統一
- [ ] 形式 MP4（H.264 / AAC）
- [ ] 音割れ・無音区間がない

## 🚦 公開前 最終QA
- [ ] サムネ（[04](04_サムネイル設計.md)）を設定／スマホ表示で読める
- [ ] タイトル（[05](05_タイトル・概要欄・SEO.md)）採用案を設定
- [ ] 概要欄・チャプター（0:00開始）を貼った
- [ ] ハッシュタグ・タグを設定
- [ ] 終了画面・カードを設定
- [ ] 再生リスト（「◯◯の選び方」シリーズ等）に追加
- [ ] 公開範囲・予約投稿の設定を確認
- [ ] 子ども向け設定／その他の規約項目を確認
- [ ] サムネとタイトルと中身が一致している（クリックベイトでない）

## 📣 公開直後
- [ ] 固定コメント（補足・関連動画リンク）を投稿
- [ ] SNS等で告知（任意）
- [ ] 自分で一度通して再生し、リンク・チャプターの動作を確認
