## このリポジトリについて
- `non-x2/non-x2` は **のんラボ（個人開発ラボ）のプロフィール兼・制作物リポジトリ**。
- 「個人がAIで何ができるか」を実証する場所。コンテンツ制作の「型（テンプレート）」が中心。
- 利用者は**非エンジニア寄り**。やり取り・ドキュメントは**日本語**、専門用語はかみ砕いて説明する。

## 作業の進め方（このリポジトリの慣習）
- **ブランチ**：`claude/<内容>` を新規作成して作業（main直編集はしない）。
- **PR**：プッシュ後にドラフトでPR作成 → ユーザー確認 → **squashマージ**。
- **コミット末尾**に以下を付与：
  ```
  Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>
  ```
- **言語**：日本語。トーンはやさしく、結論ファースト。
- **安全**：公開・削除・マージなど戻しにくい操作は事前に確認を取る。
- **トラブル対策**：過去に起きた失敗と解決策は [`docs/トラブル解決記録.md`](docs/トラブル解決記録.md) に集約。**作業前に目を通して同じ轍を踏まない**。新たに解決したトラブルは同ファイルに1件追記する。
