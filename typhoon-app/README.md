# 🌀 台風情報まとめアプリ

台風接近時にサッと開ける、静的な1ページWebアプリです。

## 特徴
- HTML1ファイルのみ（`index.html`）。サーバー・ビルド不要で、ブラウザで直接開けば動きます。
- 気象庁・NHK・tenki.jpなど**公式・信頼できる情報源へのリンク集**をまとめています。
- 台風接近前の「やることチェックリスト」付き。
- 台風の見方がわかる豆知識（進路予報円・暴風域と強風域の違いなど）を掲載。

## 使い方
`index.html` をブラウザで開くだけです。GitHub Pages等で公開する場合は、このディレクトリをそのまま配信してください。

## 注意
このアプリ自体はリアルタイムの台風進路・気象データを取得しません。最新情報は必ずリンク先の公式サイトでご確認ください。
