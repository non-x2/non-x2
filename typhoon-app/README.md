# 🌀 台風情報まとめアプリ

台風接近時にサッと開ける、静的なWebアプリです。

## ページ構成
- `index.html` — 📡 リンク集・備え。気象庁・NHK・tenki.jpなど**公式・信頼できる情報源へのリンク集**、接近前の「やることチェックリスト」、台風の見方がわかる豆知識（進路予報円・暴風域と強風域の違いなど）。
- `status.html` — 📝 今の状況。Web検索で調べた**時点でのスナップショット**（発生中の台風の有無、直近で通過した台風、今後注意すべき「台風のたまご」など）をまとめたページ。リアルタイム更新はされないため、更新したい場合はこのファイルを作り直してください。

サーバー・ビルド不要で、HTMLファイルをブラウザで直接開けば動きます。GitHub Pages等で公開する場合は、このディレクトリをそのまま配信してください。

## 注意
このアプリ自体はリアルタイムの台風進路・気象データを取得しません。最新情報は必ず各ページのリンク先の公式サイトでご確認ください。
