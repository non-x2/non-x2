# 📚 docs — Git / GitHub と Claude の使い方ガイド

非エンジニア向けに、**マージなどGit/GitHubの使い方**と、**Claude（Claude Code）への指示のコツ**をまとめた資料集です。

| ファイル | 中身 | こんな時に |
|---------|------|-----------|
| [`slides_git_claude入門.md`](slides_git_claude入門.md) | スライド（3部構成：Git基礎／指示のコツ／今回の実例） | まず全体像をつかみたい |
| [`指示書_Claudeへの指示テンプレ.md`](指示書_Claudeへの指示テンプレ.md) | コピペで使える「頼み方」チートシート | 実際にClaudeへ頼むとき |
| [`指示書_Git操作手順書.md`](指示書_Git操作手順書.md) | ブランチ〜マージの操作をステップ確認 | 操作の流れを確認したいとき |
| [`Claude最新情報_2026-06-26.md`](Claude最新情報_2026-06-26.md) | 🆕 Claudeモデル＆Claude Codeの最新アップデートまとめ（定点メモ） | 「いまのClaudeって何が新しい？」を知りたいとき |

> スライドは Marp 記法（`---` でページ区切り）でも書いてあるので、Markdownのまま読んでもOK、スライド表示にもできます。Gamma版も別途生成しています。

## 使い方のおすすめ順
1. `slides_git_claude入門.md` で用語と流れをつかむ
2. 何か頼むときは `指示書_Claudeへの指示テンプレ.md` をコピペ
3. Git操作の意味を知りたくなったら `指示書_Git操作手順書.md`

わからない用語は、いつでもClaudeに「**○○って何？**」と聞けばOKです 🙆
